Player A's Turn

Character Types and Movement Rules

1. Pawn (P):

  • Movement: Moves one block in any direction—left, right, forward, or backward.
  • Move Commands:
    • L: Move Left
    • R: Move Right
    • F: Move Forward
    • B: Move Backward
  • Example Command: P1:L (Pawn 1 moves Left)

2. Hero1 (H1):

  • Movement: Moves two blocks straight in any direction.
  • Special Ability: Can capture any opponent's character in its path.
  • Move Commands:
    • L: Move Left
    • R: Move Right
    • F: Move Forward
    • B: Move Backward
  • Example Command: H1:F (Hero1 moves Forward)

3. Hero2 (H2):

  • Movement: Moves two blocks diagonally in any direction.
  • Special Ability: Can capture any opponent's character in its path.
  • Move Commands:
    • FL: Move Forward-Left
    • FR: Move Forward-Right
    • BL: Move Backward-Left
    • BR: Move Backward-Right
  • Example Command: H2:FL (Hero2 moves Forward-Left)

Note: All moves are relative to the player’s perspective.
